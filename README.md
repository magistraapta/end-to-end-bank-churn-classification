# How to run this app
### Install all the dependencies
```bash
pip install requirements.txt
```
### Clone this repo
```bash
git clone <this repo>
mkdir <directory>
```
### Run FastAPI
```Bash
fastapi run dev main.py
```
### Run Streamlit
```bash
streamlit run streamlit.py
```
### Run MLflow 
```bash
mlflow ui
```
